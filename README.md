# 奇迹医助
“奇迹医助”是以医院为甲方开发的一款线上医疗的微信小程序。

功能包括：线上问诊、医保查询、挂号预约、一键急救（一键发送位置和电话给医院急救负责人）和乳腺癌CT图像检测。

使用的技术包括：

1. 前端：JavaScript、Vue、uni-app

2. 后端：微信云开发（使用类似于Nodejs和MongoDB）

3. 依赖 uView 样式库：https://www.uviewui.com/
